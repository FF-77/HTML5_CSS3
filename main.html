    ALGUNS COMANDOS DE HTML5 É SUAS DESCRIÇÕES.

    Código Descrição
    < !--...--> Criar comentário no HTML
    < !DOCTYPE> Define o tipo do documento
    < p> Parágrafo
    < title Título da página
    < img src=""> Carrega imagem na página
    < table> Cria tabela
    < tr> Define linha
    < td> Define a coluna
    < th> Define o texto da coluna em negrito
    < ol> Lista ordenada
    < ul> Lista desordenada
    < menu> Menu
    < li> Representa um elemnto dentro da lista ou menu

    Código Descrição
    font-size Tamanho da fonte
    color cor da fonte

    font-family Estilo do fonte

    margin-bottom Marge do rodapé
    margin-left Margem da esquerda
    margin-right Margem da direita
    width Largura
    height Altura
